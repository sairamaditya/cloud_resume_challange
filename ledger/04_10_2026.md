# Fix: CKA Badge White Space Issue and Resume Layout Improvements

## Problem

While adding the **Certified Kubernetes Administrator (CKA)** badge to the HTML resume, a large white box appeared around the image.

The issue occurred because the badge image (`cka-logo.jpeg`) contained significant whitespace inside the image canvas. When rendered in the browser, the full image (including the whitespace) was displayed, making the badge look misaligned and oversized.

Additionally, the original implementation used inline styles and an iframe for the badge display, which introduced layout inconsistencies.

---

# Solution

The fix involved:

1. Removing the embedded badge iframe.
2. Using the locally stored badge image (`image/cka-logo.jpeg`).
3. Creating a reusable **certification card layout** using CSS.
4. Applying CSS properties to properly constrain and align the image.

# Key Improvements

### 1. Removed iframe embedding

The badge iframe was replaced with a local image to avoid rendering issues and reduce external dependencies.

### 2. Image constraints

The CSS rule:

```css
object-fit: contain;
```

ensures the image fits within the defined size without distortion.

### 3. Responsive layout

Using `flexbox` allowed the badge and text to align properly across different screen sizes.

### 4. Clean component structure

A reusable **certification card component** was created using:

```
.cert-container
.cert-card
.cert-logo
```

This makes it easier to add additional certifications later.

# Lessons Learned

* Badge images often contain large internal padding.
* Using CSS (`object-fit`, fixed dimensions) can control rendering without editing the image itself.
* Reusable CSS components make the HTML easier to maintain as the resume evolves.
* Avoid embedding external iframes when a static image link is sufficient.
